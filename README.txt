API:

URL: /getNum/miaopai
Desc : get visiting number from miaopai
Method : GET
Parameters(optional) : url (http://www.miaopai.com/u/mob_49878220), there is also a testing url if you don't pass any.
Return : a list of videos with infos(num,raw_num,src)

Appendix: additional python libraries needed to be installed: selenium, flask
make adjustment according to your own environment
